# Logo Directory

Please place the Vinicius International logo file here:

- **Filename**: `vinicius-logo.png`
- **Format**: PNG with transparent background
- **Recommended size**: 200x80px or similar aspect ratio
- **Resolution**: High resolution (300 DPI) recommended for PDF quality

The logo will be used in:
- PDF headers (delivery notes and reports)
- Watermarks on all PDF pages
- Public view pages

If the logo is not found, the PDF generators will fall back to text-only headers.
